HOUSING PRICE PREDICTOR
=======================

QUICK START:
1. Double-click "launch_app.html" to open the launcher
2. Click "Launch Application" button
3. OR double-click "run_app.bat" to start directly

FEATURES:
- AI-powered price predictions
- Interactive web interface
- Feature importance analysis
- Complete model reports

SUPPORT:
- Ensure R is installed: https://cran.r-project.org
- Internet connection required for first-time package installation
